Dungeon3 - Подземелье 3 локации

Описание проекта

Dungeon3 - это консольная приключенческая игра на языке C, в которой игрок исследует подземелье, состоящее из трёх локаций. Цель игры - найти сокровища, поговорить с NPC и найти выход из подземелья.

 Игровой процесс

Игрок управляет персонажем (`@`), который перемещается по карте с помощью клавиш WASD. На пути встречаются различные объекты:

| Символ | Объект | Описание |
|--------|--------|----------|
| `@` | Игрок | Управляется пользователем |
| `#` | Стена | Непроходимое препятствие |
| `.` | Пол | Свободное пространство |
| `C` | Сундук | Содержит сокровища или ключ |
| `N` | NPC | Даёт подсказки |
| `D` | Дверь | Переход между локациями или выход |

Локации

1. **Входная комната** - стартовая локация с сундуком и NPC
2. **Библиотека** - промежуточная локация
3. **Сокровищница** - финальная локация с выходом

Цель игры

1. Найти сундук (`C`) во входной комнате
2. Поговорить с NPC (`N`) для получения подсказки
3. Перейти в Сокровищницу через дверь (`D`)
4. Найти сундук в Сокровищнице, чтобы получить золотой ключ
5. Выйти через дверь (`D`) для победы!

Системные требования

- Операционная система: Windows / Linux / macOS
- Компилятор: GCC 4.8+ или Clang
- Терминал с поддержкой UTF-8
